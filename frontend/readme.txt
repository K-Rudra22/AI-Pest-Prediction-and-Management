
# Pest Prediction Dashboard (Flutter Application)

This directory contains the Flutter-based frontend dashboard for the Pest Risk Prediction and Integrated Pest Management (IPM) Recommendation System developed for the EYIC Innovation Challenge.

The application provides a user interface that allows users to input agricultural conditions and view predicted pest risk levels along with recommendations.

## Overview

The frontend is built using Flutter and serves as the interactive component of the system. It communicates with the backend API to obtain pest risk predictions generated by machine learning models.

The dashboard presents prediction results in a clear and accessible format suitable for monitoring crop pest risk.

Flutter configuration file containing dependencies and project metadata.

## Functionality

The frontend performs the following functions:

1. Provides an interface for entering crop and environmental data.
2. Sends user input to the backend API for prediction.
3. Displays pest risk results returned by the machine learning models.
4. Presents prediction results through a structured dashboard interface.

## Technologies Used

Flutter  
Dart  
REST API communication with Python backend

## Running the Application

Install Flutter dependencies:

flutter pub get

Run the application:

flutter run

The application will start on the selected device or emulator.

## Role in the System

The frontend dashboard serves as the user-facing component of the system. It allows users to interact with the pest prediction models and view results generated by the backend prediction engine.
